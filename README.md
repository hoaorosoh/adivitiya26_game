# The Odyssey – 2D RPG Game

## Overview

The Odyssey is a 2D top-down role-playing game developed using the Unity game engine. The game follows the story of a lone individual who has crash-landed on an unfamiliar planet. Stranded far from home, the player must explore the environment, progress through different areas, and recover their damaged ship in order to return home.

This project was developed as part of the Adivitiya Game Jam.

---

## Game Objective

After a crash landing on an unknown planet, the player must survive by navigating the environment and progressing through interconnected locations. The primary objective is to locate and retrieve the damaged ship, which represents the only way to escape the planet and return home.

---

## Features

- 2D top-down movement system  
- Tilemap-based level design  
- Scene transitions using trigger zones  
- Player movement and animation system  
- Basic health and collision mechanics  
- Modular Unity project structure suitable for future expansion

## How to Play (Windows Build)

The game has already been built using Unity and does not require the Unity Editor to play.

1. Extract the provided submission folder.
2. Open the extracted folder.
3. Run the executable (⁨`.exe`⁩) file.
4. The game will start automatically.

No additional installation or configuration is required.
